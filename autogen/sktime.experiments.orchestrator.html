sktime.experiments.orchestrator module¶

class sktime.experiments.orchestrator.Orchestrator(tasks, datasets, strategies, cv, result)[source]¶

Bases: object

Orchestrates the sequencing of running the machine learning experiments.

Parameters

  • tasks (sktime.highlevel.Task) – task object

  • datasets (pandas dataframe) – datasets in pandas skitme format

  • strategies (list of sktime strategy) – strategy as per sktime.highlevel

  • cv (sklearn.model_selection cross validation) – sklearn cross validation method. Must implement split()

  • result (sktime result class) – Object for saving the results

run(predict_on_runtime=True, save_strategies=True)[source]¶

Method for running the orchestrator

Parameters

  • predict_on_runtime (bool, optional (default=True)) – If True makes predictions after the estimator is trained

  • save_strategies (bool, optional (default=True)) – If True saves the trained strategies on the disk
